# Dark Portfolio

A single-page **dark-themed portfolio** built with **HTML** and **CSS**. All images are stored in `assets/images` and linked in the layout.  

## Features

- **Hero**: Intro text, CTA buttons, and profile illustration on a dark background.  
- **Services**: Three service cards on a dark strip.  
- **Projects**: Four responsive project thumbnails.  
- **Quote & Stats**: Inspirational quote plus simple stats (projects, clients, experience).  
- **Contact**: Minimal contact form with name, email, and message fields.  

## Getting Started

1. Open the folder in VS Code.  
2. Ensure paths are correct:  
   - `index.html`  
   - `index.css`  
   - `assets/images/...`  
3. Open `index.html` in a browser or use a **live server**.  

**Note:** No JavaScript is required. Keep image filenames unchanged to avoid broken links.
